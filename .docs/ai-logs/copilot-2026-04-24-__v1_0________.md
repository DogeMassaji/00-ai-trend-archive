v1.0 全部代码已生成，文件结构如下：

```
.github/workflows/update.yml   ← GitHub Actions 定时任务（每日 UTC 0:00）
scripts/fetch.py               ← 抓取脚本
data/repos.json                ← 14 个精选 AI Coding 仓库列表
data/archive.json              ← 初始空归档 []
docs/index.html                ← 静态前端页面
docs/style.css                 ← GitHub 风格暗色样式
```

**各文件要点：**

- ：读取 `repos.json` → 调 GitHub API → 追加 `archive.json` → 输出 `today.json`，支持 `GITHUB_TOKEN` 环境变量提升限速上限，已做幂等处理（同一天多次运行覆盖而不重复追加）
- ：每日 UTC 0:00 自动运行，支持 `workflow_dispatch` 手动触发；使用内置 `GITHUB_TOKEN` 提交数据文件
- ：排行榜表格 + 点击行展示 Chart.js 折线图（近 7 天每日新增）
- ：GitHub 暗色主题风格

**首次使用步骤：**
1. 推送代码到 GitHub
2. 仓库设置 → Actions → General → Workflow permissions → **Read and write permissions**
3. 仓库设置 → Pages → Source: `main` 分支  目录
4. 在 Actions 页面手动触发一次 `Daily Fetch`，获得初始数据
